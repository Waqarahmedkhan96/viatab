# DevOps Lab - VIA Tabloid (VIATAB)

This project is part of the DOC1 course.

## Project Overview
VIATAB is a web application for managing sensational stories from VIA departments.

## Technologies
- Frontend: React / TypeScript
- Backend: Spring Boot / .NET
- Database: PostgreSQL
- Docker
- Kubernetes (Minikube)

## Run (Kubernetes)
minikube start
kubectl apply -f .
minikube service viatab-backend --url
